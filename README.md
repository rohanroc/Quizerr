# Quizerr 🎯

Quizerr is a sleek and interactive Quiz Application built using **React JS** and **Vite**. It allows users to test their knowledge across various topics with multiple-choice questions and real-time score tracking.

---

## 🚀 Features

- 🧠 **Multiple-choice questions:** Test your knowledge with a variety of questions.
- ✅ **Real-time answer validation:** Instant feedback on chosen answers.
- 📊 **Score tracking:** Keep track of your performance throughout the quiz.
- 🔄 **Play again / Reset quiz:** Easily start a new session after finishing.
- 📱 **Responsive design:** Optimized for both desktop and mobile devices.
- ⚡ **Fast and lightweight:** Powered by Vite for lightning-fast development and optimized builds.

---

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** HTML5 / CSS3
- **Language:** JavaScript (ES6+)
- **Core:** React Hooks (`useState`, `useEffect`)

## 🧩 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Clone the repository

```bash
git clone https://github.com/your-username/quizerr.git
cd quizerr
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

The application will start, and you can access it in your browser (usually at `http://localhost:5173`).

---

## 📜 Available Scripts

In the project directory, you can run:

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the app for production to the `dist` folder.
- `npm run preview`: Locally preview the production build.
- `npm run lint`: Runs ESLint to check for code issues.
